# IDEA操作Github

## IDEA连接到Github

```shell
# 1. 打开IDEA→File→settings，选择Version Control→Git，选择安装好的git路径。

# 2. 在Version Control下选择Github，添加Github账号。

# 3. 在工具栏顶部的Version Control下选择share project on→github，填写对应内容点击share。

# 4. 按流程创建token并复制内容到idea，显示登录成功并能成功拉取登录账户信息。
```
